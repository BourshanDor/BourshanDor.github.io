Cryptography Project

Advanced Encryption Standard

This project is a Python implementation of the Advanced Encryption Standard (AES) encryption algorithm, with support for both Cipher Block Chaining (CBC) and Electronic Codebook (ECB) modes.The purpose of this project is to provide a practical experience of implementing AES encryption in Python. AES is a widely used symmetric encryption algorithm, and understanding its implementation with different modes adds valuable insights into cryptography.
